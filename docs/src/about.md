# About scSignalMap
scSignalMap is an R package developed for identifying condition-specific cell–cell communication from single-cell RNA sequencing (scRNA-seq) data. By integrating differential expression analysis, ligand–receptor interaction databases, and pathway enrichment analysis, scSignalMap helps users uncover biologically meaningful signaling interactions between cell populations. The package is designed to work directly with Seurat objects and provides both modular functions and an end-to-end pipeline for analyzing cell-cell signaling. scSignalMap identifies ligand–receptor interactions across sender and receiver cell types, detects differentially expressed receptors between conditions, and links signaling changes to enriched biological pathways.
